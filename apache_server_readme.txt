1. Install Apache server 2.2 or 2.4.
2. Create  folder "\img" in Apache`s root directory.
3. Make sure that you have root access to C:\
4. Change path to your "\img" folder in  calibration-devices-system\service\src\main\resources\mail.properties
5. photo.storage.local - local path.
6. photo.storage.server - URL on server.
7. Now you can upload photos.
